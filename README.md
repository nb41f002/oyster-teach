# 臺南小蚵人雙語營 · 教學提詞台

老師上課用的提詞台（教師個人用，非學生教材）。五天課程一站到底，平板可看、可離線。

- **Day 1 勇者覺醒**（你 · 主持）
- **Day 2 食材大對決**（謝沛妤 · 神秘箱）
- **Day 3 市集英語挑戰**（謝沛妤 · 猜價格／賓果／辦桌）
- **Day 4 小小說書人**（徐威鈞 · 肢體魔法／說書草稿）
- **Day 5 傳說的勝利**（你 · 結語）

功能：五天一鍵切換、簡報截圖點圖放大、右下「四大口令」浮動速查、每天完成度勾選（存在裝置）、🌙 暗色／大字模式。

---

## 檔案結構

```
oyster-teach/
├─ index.html      ← 主程式（介面＋邏輯）
├─ data.js         ← 全部五天逐字稿內容（要改字改這裡）
└─ img/            ← 各天簡報截圖（36 張，檔名 d1_*、d2_* …）
```

> 另有一個「單檔離線版」（`教學提詞台_離線單檔.html`），圖片全內嵌，雙擊就開、斷網也能用，適合當現場備用。

---

## 部署到 GitHub Pages（跟你 0525 一樣）

1. 在 GitHub 新開一個 repo，例如命名 **`oyster-teach`**（跟現有 `oyster-camp` 分開）。
2. 把 `index.html`、`data.js`、`img/` 三個一起上傳／push 到 repo 根目錄。
3. repo → **Settings → Pages** → Source 選 `Deploy from a branch` → 分支選 `main`、資料夾選 `/ (root)` → Save。
4. 稍等一兩分鐘，網址就是：
   `https://nb41f002.github.io/oyster-teach/`

若想放在別的路徑（例如 `nb41f002.github.io/0706/`），把 repo 命名成 `0706` 即可，規則相同。

---

## 現場離線（重要）

GitHub Pages 首次載入要連網。到場前：

- 用 Safari／Chrome 開一次網址，然後 **「加到主畫面」**（iPad：分享 → 加入主畫面）。之後從主畫面開，斷網也能用。
- 或直接把 **單檔離線版** 存到平板，雙擊開啟，完全不需要網路（最保險的備援）。

---

## 之後要改內容

打開 `data.js`，每個活動是一張卡：

```js
{ id:"a1", p:"第一節 · 典禮", time:"8:40", title:"主持開場",
  img:"d1_s1",              // 對應 img/d1_s1.jpg，可省略（沒圖）
  star:true,                // 標「★ 重點」，可省略
  blocks:[
    { t:"zh", x:"中文台詞" },   // t: zh=中 en=英 cue=動作 stu=學生回 cmd=口令 note=備註
    { t:"en", x:"English line" }
  ]
}
```

改完存檔、重新 push 就更新了。想換某張簡報截圖，把新圖丟進 `img/`，改 `img:"檔名"` 即可。
